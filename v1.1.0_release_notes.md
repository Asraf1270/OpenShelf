# OpenShelf v1.1.0: The Aesthetic & Interaction Update

We are thrilled to announce **OpenShelf v1.1.0**! Following our recent milestone release, this update focuses entirely on elevating the visual design, user experience, and core interactions of the platform. We've brought a consistent, premium glassmorphic feel to every corner of OpenShelf.

## 🌟 Release Title
**OpenShelf v1.1.0: The Aesthetic & Interaction Update**

## 🚀 Key Highlights

### 🎨 Complete UI Modernization
- **Unified Design Language:** Rolled out a stunning glassmorphic interface across the platform, including the **Authentication (Login/Register)**, **About**, **Book Details**, and **Profile Editing** pages.
- **Premium Aesthetics:** Integrated a harmonious HSL-based color palette and custom typography to give OpenShelf a vibrant, modern, and state-of-the-art feel.
- **Micro-Animations:** Added fluid, scroll-triggered animations and interactive hover effects that make the platform feel responsive and alive.

### 📱 Superior Mobile-First Experience
- **Responsive Layouts:** Overhauled the Book Listing and Book Details pages. Browsing books on mobile now features an optimized two-column grid.
- **Polished Components:** Ensured all UI elements scale flawlessly across devices from desktop to mobile without losing their premium aesthetic.

### 📧 Critical Request Interactions
- **Automated Borrow Notifications:** Fixed and finalized the email notification system! Book owners will now receive beautifully formatted email alerts the moment another user requests to borrow their book.

### 🐛 Crucial Bug Fixes & Refinements
- **Profile Avatars Reimagined:** Addressed pathing issues causing broken profile pictures in the global header.
- **Seamless Profile Editing:** Fixed a persistent image upload bug in the profile editor so syncing user avatars to the JSON data store works perfectly via a single form submission.
- **Advanced Book Filtering:** Upgraded the book list to allow selecting multiple categories simultaneously for dynamic, precision searches.
- **Optimized Layouts:** Refined padding, alignments, and visual spacing for a decluttered, focused view on book covers.
