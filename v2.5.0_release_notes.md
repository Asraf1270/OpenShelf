# 📦 OpenShelf v2.5.0 Release Notes
## 🚀 v2.5.0 — Infinite Discovery & Community Support
**Release Date:** April 24, 2026

Version 2.5.0 is all about removing friction. Books load seamlessly as you scroll, navigation feels smarter and more intuitive, and the platform now gives the community a direct way to support its growth. This release is a leap forward in how users discover, navigate, and contribute to OpenShelf.

---

### 🎯 Key Highlights

#### ♾️ Infinite Scroll — Seamless Book Discovery
The books page has been completely re-engineered for effortless exploration.
- **Zero-Click Pagination**: Books automatically load as you scroll down — no more "Next Page" buttons.
- **Cursor-Based API**: Built on a high-performance cursor-based pagination system in `api/books.php` to ensure consistent results even as the catalog grows.
- **IntersectionObserver**: Uses the native browser `IntersectionObserver` API for smooth, battery-efficient scroll detection.
- **Live Search + Filter**: Infinite scroll integrates fully with the search bar and category filter — results stream in dynamically as you refine your query.
- **PDO Fix**: Resolved a critical `SQLSTATE[HY093]: Invalid parameter number` error that was causing 500 Internal Server Errors during filtered scrolling.

#### 💳 Support Us Page
OpenShelf is community-powered — now the community can give back.
- **Three Payment Methods**: Dedicated cards for **bKash**, **Nagad**, and **Rocket** with one-click number copy.
- **TrxID Submission**: Integrated transaction ID input with length and format validation so supporters can log their contributions with confidence.
- **Clean, Minimal Design**: Premium card layout with smooth hover effects and copy-confirmation feedback.

#### 🗂️ Smarter Navigation & Layout
The UI has been refined across the board for a cleaner experience.
- **Split Header**: The books page now separates the search bar from category filters. Only the category row stays sticky — search scrolls naturally with the page.
- **Redesigned Mobile Menu**: Improved hierarchy with section labels, cleaner spacing, and authentication-aware menu items that only show relevant links.
- **Support Us CTA**: A prominent "Support Us" call-to-action is now visible in the mobile navigation menu.

---

### 🛠️ Technical Improvements
- **API Hardening**: Fully corrected PDO parameter binding for search + cursor combined queries in `api/books.php`.
- **Performance**: Cursor-based pagination eliminates `OFFSET` queries, resulting in faster load times for large book catalogs.
- **Responsive Polish**: Category filter sticky bar refined for pixel-perfect alignment on mobile and desktop.
- **Code Clarity**: Simplified and documented the `IntersectionObserver` callback and API response handling in `books/index.php`.

---

### 📦 Installation & Setup
1. Pull the latest changes from the repository.
2. No database migrations are required for this update.
3. Ensure your server supports `PDO` with MySQL — no additional PHP extensions needed.
