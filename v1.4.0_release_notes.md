# OpenShelf v1.4.0: Progressive Web App (PWA) Update

We are excited to release **OpenShelf v1.4.0**! This exciting update brings native app-like capabilities to OpenShelf. Users can now install OpenShelf directly on their mobile and desktop devices for a faster, more reliable, and immersive experience.

## 🌟 Release Title
**OpenShelf v1.4.0: Progressive Web App (PWA) Update**

## 🚀 Key Highlights

### 📱 Installable App Experience
- **Add to Home Screen:** Users can now install OpenShelf on iOS, Android, and Desktop directly from their browsers.
- **Standalone Mode:** The app runs in its own window without browser UI, providing a focused, native-like experience.
- **Quick Shortcuts:** Long-press the app icon on supported devices to quickly access features like "Browse Books", "Activity Feed", and "Add a Book".

### ⚡ Smart Caching & Performance
- **Service Worker Integration:** Implemented robust service workers to manage background tasks and caching.
- **Cache-First Assets:** Static resources (CSS, JS, Logos, Fonts) are pre-cached and load instantly, significantly reducing network payload and load times.
- **Network-First Pages:** Dynamic content is always kept fresh, falling back to cached versions if the network is unstable.

### 🌐 Offline Support
- **Resilient Connections:** Core app functionality is insulated against spotty internet connections.
- **Branded Offline Fallback:** Graceful handling of network loss with a beautifully designed, glassmorphic offline page.
- **Auto-Reconnect:** The offline page automatically detects when the internet connection is restored and seamlessly redirects users back to the app.
