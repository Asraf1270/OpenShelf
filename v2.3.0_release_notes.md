# 📦 OpenShelf v2.3.0 Release Notes

**Release Date:** April 23, 2026

OpenShelf v2.3.0 is a focused release centered around streamlining the administrative experience, reinforcing data safety, and ensuring cross-browser design compatibility. 

---

## ✨ New Features & Enhancements

### 🛡️ Simplified Admin Authentication
- Removed the complex, OTP-based admin login flow.
- Implemented a fast, reliable, and secure standard email and password authentication system for administrators.

### 🔑 Admin Profile Password Management
- Added a new, fully functional "Change Password" module directly within the admin profile section.
- Administrators can now securely update their credentials without leaving the dashboard.

---

## 🛠️ System Improvements & Audits

### 💾 Backup System Audit & Verification
- Conducted a comprehensive audit of the OpenShelf backup architecture (`admin/backup.php` & `admin/backup/restore.php`).
- Verified the integrity of full database dumping and precise restoration logic, ensuring reliable data preservation for system administrators.

### 🎨 CSS Compatibility Fixes
- Addressed CSS warnings related to the `background-clip` property within the admin header (`includes/admin-header.php`).
- Defined standard, non-prefixed `background-clip` properties alongside existing `-webkit-background-clip` to ensure broad cross-browser standards compliance while preserving the intended sidebar logo visual styling.
